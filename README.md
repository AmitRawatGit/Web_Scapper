# Web_Scapper

-------------------------------------------
📌 Features
-------------------------------------------
🔗 Scrape any public webpage using URL input
🧠 Extract:
Page title & meta description
Headings (H1, H2, H3)
Paragraph content
Links (with domains)
Tables (if available)
📊 Generate insights:
Word count
Content structure
Keyword frequency
📈 Interactive visualizations using Plotly
📥 Download data as CSV / JSON
🎨 Clean, modern UI with Streamlit


-------------------------------------------
🛠️ Tech Stack
-------------------------------------------
Python
Streamlit – UI framework
Requests – HTTP requests
BeautifulSoup – HTML parsing
Pandas – Data handling
Plotly – Visualization

-------------------------------------------
⚙️ Installation & Setup
-------------------------------------------
1. Clone the repository
    git clone https://github.com/AmitRawatGit/Web_Scapper.git
    cd Web_Scapper

2. Create virtual environment (recommended)
    python -m venv venv
    venv\Scripts\activate   # Windows
    # or
    source venv/bin/activate   # Mac/Linux

3. Install dependencies
    pip install -r requirements.txt
   
5. Run the app
    streamlit run app.py

⚠️ Limitations
Some websites block scraping (e.g., LinkedIn, Amazon)
JavaScript-heavy sites may not load fully (requires Selenium)
Only works with publicly accessible pages
